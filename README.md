# 조교용 그림책자 — 클로드코드로 학과 문서 만들기

컴퓨터를 잘 몰라도 그림을 보며 1번부터 따라 하면, **엑셀·한글 학과 행정문서**(원격수업 신청서, 신규 교과목 인증서 등)를 Claude Code로 자동 작성할 수 있습니다.

> 미래융합대학 경영학전공 · AX 교육 자료

[![조교용 그림책자 표지](thumbnail.png)](https://sdkparkforbi.github.io/claude-code-booklet/)

### 👉 [그림책 바로 보기 (클릭)](https://sdkparkforbi.github.io/claude-code-booklet/)

위 이미지를 클릭하면 그림책이 열립니다. (열리지 않으면 잠시 후 다시 시도 — GitHub Pages 준비에 1~2분 걸릴 수 있어요.)

---

## 무엇을 배우나요

| 단계 | 내용 |
|---|---|
| STEP 0 | 클로드코드 + 한글(hwpx) 스킬 설치 |
| STEP 1 | 새 대화 시작 |
| STEP 2 | 작업할 폴더 선택 |
| STEP 3 | 필요한 파일 한 폴더에 모으기 |
| STEP 4 | 한국어로 말하듯 시키기 (예시 명령어 포함) |
| STEP 5 | 제출 전 검토 (소속·교수명·숫자 확인) |

## 파일

- [`index.html`](index.html) — 그림책 본문 (브라우저에서 열거나 인쇄/ PDF 저장 가능)
- `thumbnail.png` — 표지 미리보기

## 인쇄

`index.html`을 브라우저에서 열고 `Ctrl + P` → "PDF로 저장" 하면 한 장씩 깔끔하게 출력됩니다.
